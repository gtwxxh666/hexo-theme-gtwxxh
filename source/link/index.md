# 如何添加？

符合以下要求：

1. 网站强制HTTPS
2. 类型为个人博客
3. 不适用公用域名（如xx.github.io, xx.js.org）
4. 将我的网站加入友情链接中
5. 可选：备案

在评论区填写以下内容：

1. 站点名称
2. 站点链接
3. 站长头像
4. 站点描述
5. 站点截图 

图片推荐使用128x128

站点截图可选
