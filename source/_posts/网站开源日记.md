# 安装

配置好环境，在[blog-root]运行：

```Code
git clone https://github.com/gtwxxh666/hexo-theme-gtwxxh.git
```

接着，运行以下命令安装依赖：

```Code
npm install
```

运行测试：

```
hexo cl && hexo g && hexo s
```

看到效果就OK
